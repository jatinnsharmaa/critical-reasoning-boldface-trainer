# Critical Reasoning Boldface Trainer

A single-page quiz app for drilling the three roles a boldface term can play in a critical reasoning argument:

- **Non-challengeable premise** — fact, data, evidence, observation
- **Challengeable premise** — reason, justification, support, policy
- **Conclusion** — opinion, claim, prediction, position

Missed terms get re-queued so you keep seeing them until they stick. No dependencies — plain HTML/CSS/JS.
